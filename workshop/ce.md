## Custom Element

L'objectif de cet exercice est de créer un custom element.

Le custom element doit se présenter sous la forme de :

```
<boom-carrousel>
  /* contenu HTML réprésentant le carrousel */
</boom-carrousel>
```

![Carrousel](carrousel.png "Custom element")
